# Anonymous Artifact

This repository contains an anonymized implementation of MAPPER.

## Structure

- `MAPPER.py`: main entry point
- `EnvInitAgent/`: environment initialization components
- `EnvRepairAgent/`: environment repair components
- `DockerValidate/`: Docker-based validation utilities
- `tools/`: shared helper utilities
- `memory/`: memory-related modules
- `dataset/`: input datasets used by the artifact

## Notes

This repository has been anonymized for double-anonymous review.
